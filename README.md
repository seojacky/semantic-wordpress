# semantic-wordpress

Плагин для добавления семантической вёрстки в записи и страницы. Поддерживает добавление и визщуализацию тегов: &lt;article&gt;, &lt;section&gt;, &lt;div&gt; ....

Теги добавляются в HTML редакторе с помощью кнопок
